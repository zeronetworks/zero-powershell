# Get-ZNAiExclusionSourceCandidate

## SYNOPSIS
Get AI source candidates to ignore during learning.

## SYNTAX

```
Get-ZNAiExclusionSourceCandidate [-AccountName <String>] [-Limit <Int32>] [-Offset <Int32>] [-Search <String>]
 [<CommonParameters>]
```

## DESCRIPTION
Get AI source candidates to ignore during learning.

## EXAMPLES
