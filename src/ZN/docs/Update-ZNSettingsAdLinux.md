# Update-ZNSettingsAdLinux

## SYNOPSIS
Adds or update linux config for a domain.

## SYNTAX

```
Update-ZNSettingsAdLinux -DomainId <String> -ForestId <String> [-AccountName <String>]
 [-AdLinuxConfigAllowInstallPackages] [-AdLinuxConfigPassword <String>] [-AdLinuxConfigPrivateKey <String>]
 [-AdLinuxConfigUsername <String>] [-UseForAllDomains] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Adds or update linux config for a domain.
