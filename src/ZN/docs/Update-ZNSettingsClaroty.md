## SYNOPSIS

## SYNTAX

```
Update-ZNSettingsClaroty [-AccountName <String>] [-ApiUrl <String>] [-Token <String>] [-PassThru] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION

