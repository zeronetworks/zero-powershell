# Update-ZNSettingsDeploymentsCluster

## SYNOPSIS
Edit deployments cluster

## SYNTAX

```
Update-ZNSettingsDeploymentsCluster -DeploymentsClusterId <String> [-AccountName <String>]
 [-ClusterName <String>] [-PreferredDeploymentId <String>] [-Strategy <Int32>] [-PassThru] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Edit deployments cluster

## EXAMPLES

### Example 1: Update a deployments cluster
```powershell
Update-ZNSettingsDeploymentsCluster -DeploymentsClusterId C:d:1ucTOUyz -ClusterName updatedname -Strategy 1
```

This cmdlet updates a deployments cluster.
