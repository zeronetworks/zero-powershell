# ConvertTo-ZNEpochMS

## SYNOPSIS


## SYNTAX

```
ConvertTo-ZNEpochMS [[-DateTimeValue] <DateTime>] [<CommonParameters>]
```

## DESCRIPTION


## EXAMPLES

### Example 1: Convert Date to Epoch Milliseconds
```powershell
ConvertTo-ZNEpochMS 9/1/2026
```

```output
1788220800000
```

This cmdlet converts a date to epoch milliseconds.
