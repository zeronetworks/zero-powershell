# Get-ZNSettingsConnectRegion

## SYNOPSIS
List Connect Regions in Settings

## SYNTAX

```
Get-ZNSettingsConnectRegion [-AccountName <String>] [-Limit <Int32>] [-Offset <Int32>] [<CommonParameters>]
```

## DESCRIPTION
List Connect Regions in Settings
