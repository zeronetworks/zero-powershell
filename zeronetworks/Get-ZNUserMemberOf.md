# Get-ZNUserMemberOf

## SYNOPSIS
Returns a list of groups the user is a member of.

## SYNTAX

```
Get-ZNUserMemberOf -UserId <String> [-AccountName <String>] [<CommonParameters>]
```

## DESCRIPTION
Returns a list of groups the user is a member of.

## EXAMPLES

### Example 1: Get users groups

Get-ZNUserMemberOf -UserId u:a:E6iXCia4
