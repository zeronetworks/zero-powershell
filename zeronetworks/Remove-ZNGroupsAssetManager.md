# Remove-ZNGroupsAssetManager

## SYNOPSIS
Remove asset manager of group

## SYNTAX

```
Remove-ZNGroupsAssetManager -GroupId <String> -GroupOrUserId <String> -GroupType <String>
 [-AccountName <String>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Remove asset manager of group

## EXAMPLES

### Example 1: Delete an asset manager from a group
```powershell
Remove-ZNGroupsAssetManager -Groupid g:t:01445453 -GroupOrUserId g:a:cai6Wuyd -GroupType tag
```

This cmdlet deletes an asset manager from a group.
