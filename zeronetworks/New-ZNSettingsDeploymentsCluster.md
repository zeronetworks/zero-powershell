# New-ZNSettingsDeploymentsCluster

## SYNOPSIS
Create deployments Cluster.

## SYNTAX

```
New-ZNSettingsDeploymentsCluster [-AccountName <String>] [-ClusterName <String>] [-Strategy <Int32>]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIPTION
Create deployments Cluster.

## EXAMPLES

### Example 1: Create a deployments cluster
```powershell
New-ZNSettingsDeploymentsCluster -ClusterName "NewCluster2" -Strategy 1 
```

This cmdlet creates a deployment cluster.
