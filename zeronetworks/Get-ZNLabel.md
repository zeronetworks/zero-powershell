# Get-ZNLabel

## SYNOPSIS
Returns the labels.

## SYNTAX

```
Get-ZNLabel [-AccountName <String>] [-Filters <String>] [-Limit <Int32>] [-Offset <Int32>] [-Order <String>]
 [-OrderColumns <List<String>>] [-WithCount] [<CommonParameters>]
```

## DESCRIPTION
Returns the labels.

## EXAMPLES

### Example 1: List Labels
```powershell
(Get-ZNLabel).Items
```

```output
EntityCount Key   Source Value
----------- ---   ------ -----
          2 Label      1 NewValue
          1 test       1 va
          1 test       1 value
```

This cmdlet lists labels.
