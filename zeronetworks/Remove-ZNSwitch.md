# Remove-ZNSwitch

## SYNOPSIS
Delete a switch

## SYNTAX

```
Remove-ZNSwitch -SwitchId <String> [-AccountName <String>] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## DESCRIPTION
Delete a switch

## EXAMPLES

### Example 1: Delete a switch
```powershell
Remove-ZNSwitch -SwitchId w:c:jTb263az
```

This cmdlet deletes a swtich.
